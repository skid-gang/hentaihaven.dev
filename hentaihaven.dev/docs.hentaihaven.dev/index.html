hentaihaven.dev API documentation

This is meant to be used in Python. If you are using this for Python you need to import requests and random. Why do you need random? Random choices a random number between how many images my API has. For now it's only 22.

Example in Python:


import requests

import random

var = random.randint(1,22)

r = requests.get(f"https://api.hentaihaven.dev/api.json")

j = r.json()

print(j[f"{var}"])

Example in the library discord.py in Python:


import requests

import random

import discord

from discord.ext import commands

client = commands.Bot(command_prefix="?")

@client.command()

async def commandnamehere(ctx):

sauce = random.randint(1,22)

r = requests.get(f"https://api.hentaihaven.dev/api.json")

j = r.json()

embed = discord.Embed(description="Sauce provided by https://api.hentaihaven.dev/")

embed.set_image(url=j[f"{sauce}"])

await ctx.send(embed=embed)

client.run("tokenhere")
