hentaihaven.dev API documentation

This API can be utilised in any language that can provide a random number.
Why do you need random? The endpoint returns a JSON with all the 22 images, they must be indexed with an random digit.

Example in Python:

import requests
import random

digit = random.randint(1, 22)
data = requests.get('https://api.hentaihaven.dev/api.json').json()

print(data[str(digit)])

discord.py example

import aiohttp
import random
import discord
from discord.ext import commands

bot = commands.Bot(command_prefix="?")

@bot.command()
async def commandnamehere(ctx):
  digit = random.randint(1,22)
  async with aiohttp.ClientSession() as session:
  async with session.get('https://api.hentaihaven.dev/api.json') as res:
  jsonable = await res.json()

 embed = discord.Embed(description="Sauce provided by https://api.hentaihaven.dev/")
 embed.set_image(url=jsonable[str(digit)])

 await ctx.send(embed=embed)

bot.token = "put your token here"
bot.run(bot.token)
